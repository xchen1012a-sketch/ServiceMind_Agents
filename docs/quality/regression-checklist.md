# 回归清单：ServiceMind Agents

## 工单主流程

- [ ] 创建工单后状态为合法初始状态。
- [ ] 状态变更写入 `ticket_status_events`。
- [ ] 非法状态跳转被拒绝。

## Agent 运行

- [ ] 每次运行生成 `agent_runs`。
- [ ] 每个节点生成 `agent_run_steps`。
- [ ] 失败节点记录 `error_code` 和 `error_message`。
- [ ] 人工审批节点可暂停与恢复。

## RAG

- [ ] 文档版本和 chunk 可追溯。
- [ ] embedding 模型版本可区分。
- [ ] 检索命中写入 `rag_retrieval_hits`。
- [ ] 最终回答能展示引用来源。

## MCP 工具

- [ ] 工具调用写入 `tool_calls`。
- [ ] 高风险工具必须生成审批请求。
- [ ] 审批前不执行高风险写动作。

## 审批

- [ ] 审批请求、审批决策、执行结果分表记录。
- [ ] 拒绝审批不会继续执行动作。
- [ ] 审批人、理由和时间可审计。

## 评估

- [ ] 评估结果关联真实 `agent_run_id`。
- [ ] 分类、检索、工具调用、风险拦截指标可计算。

## 权限与审计

- [ ] 后端强制租户隔离。
- [ ] 后端强制 RBAC 权限。
- [ ] 关键动作写入 `audit_logs`。
